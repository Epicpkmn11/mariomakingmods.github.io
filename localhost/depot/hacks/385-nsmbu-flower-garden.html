So for the fans of New Super Luigi U out there I made this texture mod replacing the NSMBU overworld theme.

This is what it looks like on CEMU but I also tested it on Wii U and it works great:


This is my first mod and I put a lot of effort into it so I hope you'll all enjoy it.
It also perfectly goes with the original background of the game so that's good.

Features:
-new ground tileset
-one new semi-solid
-new bridge
-invisible tracks (I just think they're cool)

Download Wii U (updated):
http://www.mediafire.com/file/7vb86idu4788vqf/WU_Field_plain.szs
Put this file in your content>model folder of the game.

Issues:
-None that I know of

-AboodXD for the Miyamoto object folder
