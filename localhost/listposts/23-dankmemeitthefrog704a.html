mario maker doesnt use the sd card, so it cant read it. This applies to all games that dont use the sd card

the solution is to get a version of mocha cfw (I think its called "mocha overall sd" on the wii u homebrew app store) and start that first before starting sdcafiine. Doing so will alllow games that dont use the sd card to read the sd card and allow sdcafiine to work.

This worked in my testing with a non sd card game, mario kart 8, so it should work with all others
