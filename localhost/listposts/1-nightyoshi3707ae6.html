Here are the rules of this forum.

1) You must have a direct download link. No bit.ly stuff. Unfinished mods or Requests go in the requests forum
2) You must put full credits to the mod, regardless if what the person did is small.
3) Either a screenshot or a video will be needed to put out. If you want to surprise them, at least give them the option if they want to be surprised. As a sidenote, if it's supposed to be replicating something else, then you can show a picture of that instead.
4) One thread per mod, regardless if they are on different platforms.
If you have done a port and you have the "Super Mario Maker Modders" group, you can edit the OP to include your mod.
If you have done a port and you do not have the "Super Mario Maker Modders" group, you can make a post, and wait for a moderator/the person who made the thread to add it in
5) Use the tags! If you've done a character mod, a theme mod, or another type of mod, specify it. If your mod is wii u exclusive, specify it.

Any threads that do not comply will be put in the Requests forum

Command line for import:
