Credits to Tardanas.



In this example you see two blocks.

  • Every block has 32 bytes
  • Placed blocks start at address 0x1B0 and end at address 0x145F0
  • Before that address is data about start and finish
    • TODO
  • Yellow are x and y coordinates of type uint_16
    • The actual values can be calculated by x = (readUInt16_BigEndian(OFFSET_OF_2) - 0x50) / 0xA0 and y = (readUInt16_BigEndian(OFFSET_OF_8) - 0x50) / 0xA0
  • Grey might be some kind of z-Index
    • I tried to change bytes directly but couldn't find any difference
  • Green is x and y dimension of type uint_8
  • Purple is the block's type of type uint_16
  • Cyan is either 0x00 or 0xFF
    • if 0x00: refers to any block with dimension 1,1
    • if 0xFF: refers to any block that can change its dimension
  • Black is always the same for every placed block
  • Byte 0x17 sometimes has different values, but changing it had no effect
  • Any other byte is zero

The angry sun is an enemy that made an appearence in 1 game, which is SMB3, and in only one level. While Newer SMBW brought it back, you can actually replicate the functionality. One problem is, though, is that it doesn't come down at you like the actual game, but it does "rumble".

1. Place either a podaboo or a spiny in your level.
2. Place a mushroom on the podaboo/spiny.
3. Place wings on the podaboo/spiny.
3. Put the podaboo/spiny in a cloud.
4. Enjoy the "Angry Sun"

If you make the cloud and wings invisible and replace the podaboo with an image of the angry sun, it could actually look like if there is a real angry sun.

So this mega block will take up 4x4 tiles. It'll even make the mushroom end up on the top and not on the side. The only downside to this mega block is that you can only hit it from below.

Firstly, on the top left 2x2 box, you'll want to add wooden blocks.
On the bottom left 2x2 box, on the very left tiles, you'll want to add wooden blocks.
On the top right tile of that same 2x2 box, you'll want to add a shellmet.
On the bottom right tile of that same 2x2 box, you'll want to add a question block with a sideways spring inside.
On the bottom right 2x2 box, you'll want to add wooden blocks to every tile except for the top left tile, which will be your item.
on the top right 2x2 box, you'll want to add a one way platform, and on top of that, the right tiles of that box with sideways springs, and the left tiles with cloud platforms.

If you want the item to always go to the right, you'll need to add a sideways spring on top of the top left tile.
