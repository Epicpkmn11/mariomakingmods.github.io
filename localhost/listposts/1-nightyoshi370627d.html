What You Need!

-Windows (Linux Guide will come soon)
-The latest SMM hack pack

Let's start

1. Extract the contents of SMM.zip to your C:\SMM\
2. Copy the .szs file that you want to extract to that folder
3. Rename the .szs file to extract.szs
4. Run decomp.bat. A bunch of files & folders will be created, but you'll only need to focus on the "1_FTEX" folder.
5. Now drag that .gtx file inside the "1_FTEX" folder onto gtx_extract.exe made by MasterVermilli0n.

You're now done. The .dds file has now been into your hands, and you can edit it using GIMP.
