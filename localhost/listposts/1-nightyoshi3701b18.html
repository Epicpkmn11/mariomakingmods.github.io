What You'll need
  • The latest version of Super Mario ReMaker
  • The ability to load up exe files.
    • If for some reason your PC won't open .exe files, follow this tutorial

How to install the game
  1. Extract the SuperMarioRemaker.zip file.
  2. Run the SuperMarioReMaker.exe file
    • A warning should pop up, saying that you need to install a mod. That is normal.
  3. Close the game
    • The purpose of loading the game is to make the AppData folders generate
  4. Install both a custom character and a custom theme. I'll make tutorials later on for both of those.
  5. Now Run the SuperMarioReMaker.exe file again.

If all goes well, you shouldn't see any errors. Now go enjoy the game!
