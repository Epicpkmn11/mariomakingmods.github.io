 		What you need

- ctpktool
- Windows

1. Download and extract the zip file.
2. Place your ctpk file in the same directory.
3. Open command line to that directory
4. Type the following (with replacing "file.ctpk" with the file name):

ctpktool.exe -x file.ctpk

5. Now you can do your edits to the generated PNG without changing the resolution. Also, do not touch the XML file unless noted.
6. Once you are done with the edits, type this (with replacing "folder" with the folder that was made):

ctpktool.exe -c folder

7. Enjoy
