There are also some new site updates.


First off, we now have a new WYSIWYG postbox, which will basically allow you to see your posts what they will look like, without those annoying tags. Of course, this wont work with post layouts, in which itll work with the "preview" button.

Secondly, Ive developed something for Android users: A mobile web app. Basically, the difference is, that there is a splash screen, a custom theme, and there is no top header (where the URL would normally show). This exists on iOS, but I did that a long time ago, and it only removes the URL header.

Thirdly, the site should see a significant improvement in performance. I switched us over to HTTP2, and I removed a bunch of scripts that would slow us down, that wasnt really used. Thanks Google Lighthouse.
