Ok, so recently, there has been many misunderstanding on how bans work, and what the rules are. So here are some clarifications.

Firstly, Mario Making Mods is a network. Basically, it doesn't matter if it happens on the Discord or forum. If you act like an idiot on one of them, expect a ban on the other platform.

Secondly, if you joined the network just for drama, expect a ban, and quickly. I'm not going to tolerate any drama here.

Thirdly, bans are never permanent. They are also rarely a set amount of time. If it were all permanent, it would be unfair, meaning you can never change. Same goes for timed bans, because then you can wait it out.

Those who are caught massdeleting their own posts, will be Banned, and have all their posts undeleted.

Finally, if you are currently banned, there is a way to unban. Prove to me that you have changed. How? Simple. You can go join other communities, and show your improvement there. If we feel you have changed, you'll get an unban.

However, there can be some exceptions. For example, if you are a spam bot, you are banned permanently, without any excuses.

I hope I make this quite clear.
