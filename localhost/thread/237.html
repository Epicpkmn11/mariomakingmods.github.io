What you'll need

- ManiaModStudio
- Python 2.7

1. Extract the files in the ManiaModStudio/Tools folder to the same folder that Sonic Mania is.
2. Make a backup of the sonic mania data.rsdk file.
3. Drag the data.rsdk file onto the "rsdkv5_extract.py" file.
4. Enjoy modding Sonic Mania...
