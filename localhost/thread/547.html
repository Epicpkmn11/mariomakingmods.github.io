 		I have no experience with Utau, but it seems to be a Vocaloid knockoff (or something like that). People make music using these "Vocaloids", which are vocal synthesizers that use real human voices for each and every vowel / consonant sound, to replicate human singing. Sega has released several Vocaloid games, known as "Project Diva". There is a Project Diva for the PS4 that starts off as try before you buy, and if you like it, you can actually buy two song packs for real cheap. The song packs cover *several* Project Diva games, so it's actually a pretty great deal!
