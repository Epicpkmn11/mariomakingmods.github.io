SMBNext: Halloween Special DX & SMBNext: Demo are out now!
The downloads and guide is down below.




- Fun new levels
- Brand new world themes
- Fresh soundtrack with some returning tracks
- New and returned enemies and level mechanics




Super Mario Bros. Next is a non-profit, fan created modification for New Super Mario Bros. 2.
We aim for quality, and wish to release a game that will be as enjoyable as any other NSMB game, if not better.

This project is in active development, and the team is hard at work producing new content for the community.
This thread will be actively updated as development progresses.













NOTE: If you have already played other NSMB2 mods, make sure
to delete the Luma directory for your NSMB2 TitleID before proceeding.

1. If it isn't already, set your 3DS up with Luma3DS by following the 3DS hacking guide here.

2. Enable game patching by holding SELECT on 3DS boot, ticking the "Game patching" box, and pressing START.

3. Download the version of SMBNext you want to play using the links below:

SMBNext: Halloween Special DX (Rev. 7): Google Drive | MEGA

SMBNext: Demo (Rev. 2): Google Drive

NOTE: THE INSTRUCTIONS BELOW ARE ONLY REQUIRED FOR HALLOWEEN SPECIAL DX & DEMO

4. Insert your 3DS SD card into your PC, and find your TitleID version from the list below:

NSMB2 Standard Edition:

EUR - 000400000007AF00

USA - 000400000007AE00

JPN - 000400000007AD00

KOR - 00040000000B8900


NSMB2 Gold Edition:

USA - 0004000000137E00


NSMB2 Special Edition:

EUR - 0004000000137F00

JPN - 0000400000137D00

5. Extract the downloaded mod, and copy the "romfs" folder to SD:/luma/titles/<Your NSMB2 TitleID>/.

Replace <Your NSMB2 TItleID> with the one you picked from above.
Example directory (for USA Gold): SD:/luma/titles/0004000000137E00/romfs/Course, Unit, etc

6. Turn on your 3DS, with the SD Card inserted, and open up your version of NSMB2.

NOTE: The selected version of SMBNext will persist until you remove or rename the "romfs" folder.
