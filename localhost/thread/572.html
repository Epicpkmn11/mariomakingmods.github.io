So basically, I have made a new site layout for all of us to enjoy! It is called "Modern Mario Making Mods Layout", and it's accessible by going to Edit Profile -> Site Layout, and I do think it takes a much more modern take on the site layout. It makes it so that way the header always sticks to the top but the logo is just the text, without the mario's. This layout was suggested by tee, so huge shout-out to him. But as I started using it more and more, I've really started to dislike the old one, with that huge bulky logo. In fact, in our Github code, the old Default Layout isn't even a thing. In the new version of this board (and the current one), this is causing some serious bugs. So I'm wanting to hear your opinions: Should we get rid of the old one, or add it as another option (like it is now)?

Also, if you want to have a different site layout, please specify in the thread below! We'd love to see your ideas and suggestions on what layout you'd like to see. Thank you for your feedback, and I hope we can make the site easier for you. Bye!
