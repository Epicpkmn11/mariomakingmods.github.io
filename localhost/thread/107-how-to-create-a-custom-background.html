https://mega.nz/#!WdZmlAgQ!xhGSSQqwoU15VpFkAZvv9HnA7MdyCdz3aYxlPKNJVaY

Or, you could use an existing BG and texture edit it.

You can try replacing the models of other backgrounds, but I'm not exactly sure how. :P

This tutorial will focus on giving your background its own unique ID/slot.
You first need to name your background. For example, bgA_1111 for a Scenery
bg. Or, bgB_2354 for a backdrop bg. It doesn't matter too much. It just needs
to correspond with the 4 # pattern.

You also want to make sure the bgA or bgB makes sense. Meaning, it's bgA if it's a Scenery bg, and
it's bgB if it's a Backdrop bg. Name the file with your new ID, but don't forget the original one.

After you have all that sorted out,

open the background's arc file in a Hex Editor. I
suggest using HxD.

Now, what you need to do is replace the original file name of the BG with your new ID.
The original file name doesn't only appear once. It will appear multiple times at
different parts of the arc file.

-There are 2 ways of doing this. You could scope through the Hex Editor and replace each
time the name shows.

-Or, you could use HxD's "Replace" option and do it that way.

open the arc file in BrawlBox. Any version of it is fine.

Change each time the original file name appears to your new file name. It should appear about 2 or 3 times.

-The brres name
-the model name
-the animation texture name

It may appear at other places, but I'm not sure.

Once you've finished that, save and close the arc file. That's it! You're done!
