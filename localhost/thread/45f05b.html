Basically, we have a depot...sort-of. See, I made it so that way basically, threads in the "Project Releases" forum can have Download links, screenshots, etc... Lets go over what a depot needs to have so it can be a depot.

~Reviews? Yep (star posts/favorite threads)
~Comments? Yep
~Screenshots? Check.
~Download link? Check.
~Ability to filter....yes?
~Upload directly onto the site? Depends if its under/over 10 MB

Well, in any case, we've met all the requirement, so I guess we can consider we have a depot. Mission accomplished.
